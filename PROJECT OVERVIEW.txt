### **PROJECT OVERVIEW** **App Type:** Mileage Tracking App for iOS **Target Users:** Freelancers and self-employed individuals **Core Value Prop:** "Anti-feature" app - fewer features, zero learning curve, does essentials perfectly **Timeline:** MVP in 1-2 months **Development:** Self-developed using Cursor (AI-assisted coding) --- ### **MVP FEATURE SET** #### **Must-Have Features (v1.0):** ? **One-tap trip start/stop** (manual tracking only) ? **Trip categorization:** Business, Personal, Medical, Charity ? **IRS-compliant mileage rates** (2025 rate pre-loaded) ? **PDF report generation** with header "Business Expense Mileage Report" ? **Local device storage** (no cloud/user accounts in MVP) ? **Clean, minimalist UI** - professional and friendly #### **Deferred to Post-MVP:** ? Automatic trip detection ? Offline sync / cloud backup ? Photo receipt attachment ? Calendar integration ? Favorite locations ? Monthly notifications --- ### **DETAILED FEATURE SPECIFICATIONS** #### **1. Trip Tracking** - **Start Trip:** Single tap button to begin tracking - **End Trip:** Single tap to stop and save - **Data Captured per Trip:** - Start time & location (GPS coordinates + address) - End time & location - Total distance (calculated, not odometer) - Optional notes/purpose field - Trip category #### **2. Trip Categorization** - **Categories (most common options):** - Business (default) - Personal - Medical - Charity/Charitable - **Settings:** Allow user to set default category #### **3. IRS Mileage Rates** - **2025 Rate:** Pre-loaded (you'll need to confirm current IRS rate - likely $0.68-0.70/mile) - **Manual Override:** Allow users to adjust rate if needed (for past years or special cases) #### **4. PDF Report Generation** - **Report Header:** "Business Expense Mileage Report" - **Information Included:** - Trip date - Start location - End location - Miles driven - Category - Mileage rate ($/mile) - Total deduction amount (miles ? rate) - **Report Period:** User selectable (month, quarter, year, custom date range) - **Export:** Share via iOS share sheet (email, files, etc.) #### **5. Settings/Configuration** - **Default trip category selection** - **Measurement units:** Miles (km for future international expansion) - **Mileage rate adjustment** --- ### **PLATFORM & TECHNICAL** - **Platform:** iOS only (iPhone) - **Development:** Cursor AI-assisted development - **Data Storage:** Local device storage (CoreData or similar) - **GPS:** iOS Location Services - **PDF Generation:** iOS native PDF rendering --- ### **MONETIZATION (MVP)** **Freemium Model - Day One:** - **Free Tier:** - 40 trips/month - Manual tracking - Basic PDF reports - **Premium Tier:** $4.99-9.99/month - Unlimited trips - Cloud backup (post-MVP feature) - Advanced reports (post-MVP) **Implementation:** Apple In-App Purchase (IAP) --- ### **DESIGN PRINCIPLES** - **Aesthetic:** Minimalist, clean, professional, friendly - **Onboarding:** Jump right in with contextual tips (no lengthy tutorials) - **UX Goal:** Zero learning curve - **Interface:** Uncluttered, single-purpose focus - **Inspiration:** Set-and-forget automation (for future versions) --- ### **USER FLOW (MVP)** 1. User opens app 2. Sees simple home screen with large "Start Trip" button 3. Taps start ? GPS tracking begins 4. While driving, can add notes or change category 5. Arrives at destination
